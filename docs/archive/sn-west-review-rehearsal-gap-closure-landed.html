Residual after gap-closure

Of the 50 non-skipped sources not yet accepted:

  • 37 carry a correct, valid produced name awaiting review acceptance (25 pending, 8 drafted, 4 reviewed) — e.g. line_integrated_electron_density_of_line_of_sight, total_energy, toroidal_plasma_beta, variation_of_length_of_interferometer_beam, initial_wave_polarization_ellipticity_at_line_of_sight. These are NOT naming gaps — the names exist and validate; they are a review-entry gap: names produced at name_stage=pending (score null) are never claimed by REVIEW_NAME (which claims drafted), so the pool reports no_eligible_work while valid names sit unreviewed. Advancing them to accepted is a review pass (stage-2 prep), not composer work.
  • 3 superseded — beta_tor variants whose source should re-link to the successor normalized_toroidal_plasma_beta (a provenance relink).
  • ~9 genuinely unnamed — the hard residual: the flux-surface metric coefficients gm1 (⟨1/R²⟩), gm6, gm7 (⟨|∇ρ|⟩), gm9 (deep nested inverse/square or a |∇ρ| carrier the composer still won't assemble); tf/delta_b_field_phi_vacuum_r (change_in not applied); ece/channel/optical_depth (should reuse opacity); the contour-tree node; h_98 (confinement-scaling factor); and the hard-x-ray peak position. (signal_to_noise now composes from the new signal_to_noise_ratio base — quarantined pending a validation follow-up, no longer a naming gap.)

The starting plateau (operator-composed + line-of-sight + genuine-token sources unnameable) is closed: every one of those classes now composes a correct name. What remains is a small, well-understood tail — a review-entry throughput matter (stage-2 prep) plus ~10 esoteric or skippable quantities.
